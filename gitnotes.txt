
Data type in python defines the type of value a variable can hold. Different types of data types are int, float, bool, str, complex.




int: represents whole numbers ex: 1,2,10,100.
float: represents decimal numbers also known as continuous. ex:3.14
complex: represents complex numbers ex: 2+3j, 4-5j.
boolean: represents true or flase values ex: True, False
string: represents group of characters ex:"swamy", "reddy".
